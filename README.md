# TokenTree Formatter

Formatter based on tokentree library

## Features
- Indentation
- Whitespace
- Wrapping
- EmptyLines
  - abstract
  - enum abstract
  - class
- SameLine
  - if
  - loops
  - try / catch
- LineEnds
- `hxformat.json` config file
